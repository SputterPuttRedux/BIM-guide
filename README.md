# BIM-guide
A single-page travel guide to Barbados built for a friend visiting the island. Browse curated spots across three categories — outdoors &amp; nature, food &amp; bars, and beaches — with insider notes, embedded Google Maps, and one-tap directions. Light/dark mode included. No frameworks, no dependencies, just HTML.

View it live [here](https://sputterputtredux.github.io/BIM-guide/)
